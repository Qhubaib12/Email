# Lead Finder

A clean web tool for searching email and profile information.

## Quick start

1. Open the app.
2. Choose a search type.
3. Enter details and run search.
4. Review results in the formatted results panel.

## Deployment notes

- API functions use a versioned runtime builder in `vercel.json`.
- Node is pinned to `20.x` in `package.json` and `.nvmrc`.
- If deployment still reports Node `24.x`, set your project Node version to `20.x` in hosting project settings.
